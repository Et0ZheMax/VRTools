# Viking Rise Tools (MVP)

MVP веб-проект с инструментами для игроков Viking Rise:
- калькулятор ресурсов;
- калькулятор ускорений;
- калькулятор ферм;
- планировщик развития;
- база героев (sample data);
- заготовки под гайды и SEO-страницы.

## Запуск

```bash
npm install
npm run dev
```

Открыть: http://localhost:3301

## Стек

- Next.js App Router
- TypeScript
- Tailwind CSS
- lucide-react
